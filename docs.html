Usage:

Inside the sheet portal edit or create the Tag with the name of a listening event

If you intend to use the bot as a button you're best off with the onClick tag

If you have knowledge of JavaScript, onClick is essentially a callback

e.g Everytime a bot is clicked the onClick is fired for the bot that was clicked and that bot only

Inside the text editor for the listening event you chose, write the code for the button!

Creating More Buttons:

To get more buttons: clone an existing button using the following:

Create a variable named ButtonLabel and assign it to the label of the button you would like to change. Make sure you wrap the label in "" (double quotes) so it is in the form of a string

const ButtonLabel = "REPLACE WITH BUTTON LABEL";

Call the create() function

create(getBot("label",ButtonLabel),{color:"black"});

Note

One line example

create(getBot("label","REPLACE WITH BUTTON LABEL"),{color:"black"});
